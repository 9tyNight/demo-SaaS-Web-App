# Architectural Compliance SaaS Prototype

A Vite, React, and Tailwind CSS prototype that simulates a multi-tenant architectural compliance review workflow.

## What It Demonstrates

- Tenant-aware organization switching
- Architectural PDF upload dropzone
- Five-second async processing simulation for a Python/Celery/Redis engine
- Interactive compliance findings and detail review panel

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
